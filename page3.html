Expériences professionnelles

Avril – Juin 2025 — Stage chez Noemis

Noemis – IP Technologies and Human Inside, Clermont-Ferrand (63)

Noemis est une entreprise spécialisée dans les solutions de vidéosurveillance et les infrastructures réseau. Le pôle service dans lequel je travaille intervient régulièrement pour configurer des ordinateurs destinés à la gestion de caméras ou au stockage de données vidéo. Ces machines, livrées vierges (sans système d’exploitation), doivent être configurées selon les besoins des clients (OS spécifique, configuration réseau, sessions multiples, etc.).

Sujet : Amélioration du système de déploiement MDT/WDS

Le stage consistait à optimiser le serveur MDT/WDS existant, principalement utilisé pour le déploiement de Windows 10, afin de le rendre compatible avec de nouveaux environnements comme Windows 11, AlmaLinux et Windows Server 2025. Les scripts de configuration existants pour Windows 10 n’étaient pas compatibles avec Windows 11, qui se limitait à l’installation de quelques applications (Chrome, VLC, Notepad++) sans automatisation poussée.

Travaux réalisés :

  • – Suppression des applications non professionnelles (jeux, Xbox, etc.)
  • – Personnalisation de la barre des tâches et de l’explorateur de fichiers
  • – Automatisation d’actions via la simulation de touches clavier (scripts VBS)
  • – Intégration de nouveaux systèmes d’exploitation dans le serveur MDT/WDS, dont AlmaLinux

L’objectif est d’accélérer les installations tout en fiabilisant la configuration des machines livrées aux clients.
